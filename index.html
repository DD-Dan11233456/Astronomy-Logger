How to Use This Logger

  1. Configure Session Preferences: Type your target tracking location and select a time zone preference before locking configurations.
  2. Enter Celestial Data & Notes: Log your target coordinates, using the optional Object Observation Notes field for individual row commentary.
  3. Re-edit Rows: Hit a row's red Edit button to fix typos. The individual timestamp will update automatically.
  4. Utilize Note Panels: Use the top text field for environment parameters, and the large purple Post-Session Summary area for multi-line freeform notes of any kind.
  5. Export Your Log: Click Finish & Download TXT File to build a clean spreadsheet layout containing your dynamic table framed by your setup metadata and ending remarks.
